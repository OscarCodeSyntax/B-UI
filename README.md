# LM-UI

**To Run the ui:**
```
npm run start
```

**To format the ui:**
```
npm run pretty
```

**To test the ui:**
```
npm run test
```

**To clear and reset node conflicts and dependencies:**
```
 rm node_modules
 rm package-lock.json
 npm cache clean --force
```
